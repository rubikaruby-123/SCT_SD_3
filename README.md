
# 🎮 Sudoku Master

A modern and feature-rich Sudoku game developed using **Python** and **CustomTkinter**. Challenge yourself with multiple difficulty levels, smart assistance tools, and an attractive user interface.

## ✨ Features

* 🧩 Supports **4×4, 9×9, and 16×16** Sudoku boards
* 🎯 Multiple difficulty levels: Easy, Medium, Hard, and Expert
* 📅 Daily Challenge mode
* 💡 Hint system for solving tricky cells
* ⚡ Automatic Sudoku solver
* ✔️ Solution validation and mistake detection
* ❌ Three-strike mistake limit
* ↩️ Undo and ↪️ Redo functionality
* 💾 Save and Resume game progress
* ⏱️ Built-in timer with Pause/Resume option
* ⌨️ Keyboard shortcuts for quick actions
* 🎨 Beautiful gradient background and modern UI

## 🛠️ Technologies Used

* Python 3
* CustomTkinter
* NumPy
* Tkinter
* JSON

## ⌨️ Keyboard Shortcuts

| Shortcut | Action    |
| -------- | --------- |
| Ctrl + Z | Undo      |
| Ctrl + Y | Redo      |
| Ctrl + S | Save Game |
| Ctrl + H | Get Hint  |

## 🚀 Installation

```bash
pip install customtkinter numpy
```

Run the application:

```bash
python main.py
```
